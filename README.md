This is project for EE409 AI and Data Analytics in Power System which is one of subject in Electrical Engineer Department at Faculty of Engineering in Chiang Mai University

How to Install
1. Clone this repository
2. Run command: pip install -r requirements.txt

To start the program.<br>
Run command: python predict.py<br>

To create model.<br>
Run command: python train.py<br>
